Free live chat widget for your site

== Installation ==

This section describes how to install the plugin and get it working.

1. Download and extract the tawkto2.3.x.zip

2. Copy all the required files following the path in the included catalog directory.

3. In the Admin backend, Under the Modules sidebar menu, click on Boxes to view the list of available modules.

4. Click on the Install Module button, look for the tawk.to module and install it.

5. Click on the Edit button to edit the module config. Enter the tawk.to widget script in the Widget Script text input and save.

Note: You will need a free tawk.to account to generate the embed code, create one here : https://tawk.to/?utm_source=oscommerce&utm_medium=link&utm_campaign=signup
